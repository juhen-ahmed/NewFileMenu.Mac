# NewFileMenu

New File on Mac? Easy, NewFileMenu makes it's easy

NewFileMenu is a finder extension which allows creating a new file in a simple & easy way.

NewFileMenu works fine on latest MacOS Tahoe 26.4.1!

## Feature:

1. New file easily.
2. Spotlight like pop-up.
3. Supports common files (".html", ".css", ".txt", ".js", ".json", ".md", ".py", ".sh", ".xml", ".yaml", ".yml", ".swift", ".java", ".cpp", ".c", ".ts", ".jsx", ".sql", ".php").

## Important Note / How to enable:

1. Go to 'System Settings > General > Login Items & Extensions > App Background Activity > Extensions > NewFileMenu' set to enable.
2. Go to 'System Settings > General > Login Items & Extensions > Open at Login' add NewFileMenu.
